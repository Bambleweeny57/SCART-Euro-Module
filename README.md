# SCART-Euro-Module
Euro Module faceplate for a SCART connector - 75mm

I designed this for my Retro Computing Workbench after not been able to find one off the shelf.
I added a Scart Female connector with self tapping screw points from Rapid Electronics, but you casn find them in most places in the likes of Ebay.
T then took a standard 1.5m Scart Lead, removed one and and soldered and shrinked each of the 21 wires which was a long job, but makes for a neat AV solution on my Workbench.
